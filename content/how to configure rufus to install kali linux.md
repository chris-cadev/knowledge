#os/linux #bios
1. open rufus
2. select you kali image
3. select your device (USB)
4. partition scheme as "MBR"
5. Target system as BIOS or UEFI
6. label it
7. file system as FAT32 (default)
8. cluster size as 8192 bytes (default)
